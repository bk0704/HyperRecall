## One-Click Access Methods Comparison

┌──────────────┬───────────────┬──────────────────┬────────────────┐
│   Method     │   Command     │   Setup Time     │   Best For     │
├──────────────┼───────────────┼──────────────────┼────────────────┤
│ run.sh       │ ./run.sh      │ 0 seconds        │ All users      │
│ Makefile     │ make run      │ 0 seconds        │ Developers     │
│ Desktop Icon │ Click icon    │ 5 seconds*       │ Desktop users  │
│ Aliases      │ hr            │ 1 second**       │ Power users    │
│ Manual       │ 4+ commands   │ N/A              │ Advanced users │
└──────────────┴───────────────┴──────────────────┴────────────────┘
  *One-time: run ./install-launcher.sh
  **Per session: source aliases.sh

## File Organization

HyperRecall/
├── Scripts (One-Click Access)
│   ├── run.sh              ← Main launcher (Linux/Mac)
│   ├── run.bat             ← Main launcher (Windows)
│   ├── run.ps1             ← Main launcher (PowerShell)
│   ├── Makefile            ← Make targets
│   ├── dev.sh              ← Development build
│   ├── aliases.sh          ← Shell shortcuts
│   ├── check-setup.sh      ← Setup verification
│   └── install-launcher.sh ← Desktop integration
│
├── Documentation (Getting Started)
│   ├── QUICKSTART.md       ← Start here! ⭐
│   ├── USAGE.md            ← Complete guide
│   ├── ACCESS_METHODS.md   ← All methods overview
│   ├── TROUBLESHOOTING.md  ← Problem solving
│   └── ONE_CLICK_SUMMARY.md← Implementation summary
│
└── Source Code & Assets
    ├── src/                ← Application code
    ├── assets/             ← Fonts, icons, themes
    ├── external/           ← Dependencies (raygui)
    └── CMakeLists.txt      ← Build configuration


## Quick Decision Tree

    Need to run HyperRecall?
            │
            ▼
    ┌───────────────────┐
    │ First time user?  │
    └────┬──────────────┘
         │
    Yes──┼──No
         │  │
         │  └─→ Just run: ./run.sh ✓
         │
         ▼
    ┌─────────────────────┐
    │ Install deps first? │
    └────┬────────────────┘
         │
    Yes──┼──No
         │  │
         ▼  └─→ Run: ./run.sh
    make install-deps       (will check and report)
         │
         ▼
    ./run.sh ✓


That's it! The goal was to make it as simple as possible.
Just one command: ./run.sh (or run.bat on Windows)
